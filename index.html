SLThree

SLThree (англ. Script Language 3 — Скриптовый язык 3, "третий скриптовый") — язык программирования, базирующийся на платформе .NET Framework и тесно с ней интегрируемый.

Особенности

  • [В разработке] JIT-компиляция в .NET Framework IL (который впоследствии подхватывает JIT платформы)
  • Опциональная типизация, типы указывать можно, но не обязательно
  • Будучи динамическим, язык использует не DLR, а собственное архитектурное решение

Почему не DLR

Исходя из данной статьи, DLR при работе с COM-типами (для межязыкового взаимодействие того же IronPython и C# используется COM API) может вызывать утечки памяти при передачи dynamic'ов. Кроме того, организовать ту же опциональную типизацию с DLR было бы невозможно.
